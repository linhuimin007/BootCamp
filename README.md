# BootCamp
#Name: 
Task management programme

#Description: 
a program for a small business that can help it to manage tasks assigned to each member of the team

#A table of contents:
Users can operate:
r - Registering a user
a - Adding a task
va - View all tasks
vm - view my task
d - display statistic
e - Exit

#how to install
You can just run it in VS code

#How to use this project
Make sure your have two .txt file
one is called user.txt, the default name and password is: admin, adm1n
one is called tasks.txt, it contains the contents:admin, Register Users with taskManager.py, Use taskManager.py to add the usernames and passwords for all team members that will be using this program., 10 Oct 2019, 20 Oct 2019, No
